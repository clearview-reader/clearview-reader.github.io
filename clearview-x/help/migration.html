Migration Steps

Open Clearview X, and import your book folders (the same folders as in Clearview) to library.

Close Clearview X

From Clearview data directory (~/Library/Containers/com.canoejoy.Clearview/Data/Library/Application Support/Clearview), copy the following files and directories to the Clearview X data directory(~/Library/Containers/com.canoejoy.ClearviewX/Data/Library/Application Support/Clearview)

Cache, Covers, Notes, History.plist, books.db, cloud.db

Open Clearview X again, now you will get back all your collections and notes.
